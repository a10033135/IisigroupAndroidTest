### 規格說明
- 此為設定頁面，控制整個APP及各頁面的功能
- 畫面分為二個區塊
    - 標題 TopAppBar：「設定頁面」
    - 內容
        - 條列式各個項目
        - 版本號碼：APP的版本號碼
        - 暗黑模式：以 switchBar 控制是否為暗黑模式
        - 自動同步：以 switchBar 控制是否開啟自動同步，控制 exchangeRate / flight 頁面
          - 自動同步的秒數控制：當開啟自動同步才能選擇，預設為 10 秒，選項：10秒、1分鐘、5分鐘
        - 貨幣代碼：點擊時顯示 exchangeRate 預設的貨幣代碼，初始值為 TWD ，控制 exchangeRate 的預設貨幣
