### 功能說明
- 此為設定頁面，控制整個 APP 及各頁面的功能設定
- 支援暗黑模式切換、自動同步控制、貨幣代碼偏好設定

### 頁面內容
- 標題 TopAppBar：「設定頁面」
- 內容（條列式）：
  - 版本號碼：顯示 APP 版本號碼
  - 暗黑模式：以 SwitchBar 控制是否啟用暗黑模式
  - 自動同步：以 SwitchBar 控制是否開啟自動同步，控制 exchangeRate / flight 頁面
    - 自動同步秒數：開啟自動同步後才能選擇，預設 10 秒，選項：10 秒、1 分鐘、5 分鐘
  - 貨幣代碼：點擊時顯示可選擇的貨幣代碼，初始值為 TWD，控制 exchangeRate 的預設貨幣

### 技術文件
- 儲存方式：使用 DataStore 保存設定，預設值建立於 Constants 物件統一管理
- 各設定項目預設值：

| 設定項目 | 預設值 |
| :--- | :--- |
| 暗黑模式 | 關閉 |
| 自動同步 | 開啟 |
| 自動同步秒數 | 10 秒 |
| 貨幣代碼 | TWD |
