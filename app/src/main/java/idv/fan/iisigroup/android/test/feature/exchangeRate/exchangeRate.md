### 規格說明
- 此為匯率計算機
- 畫面分為三個區塊
    - 標題 TopAppBar：「航班資訊」
    - 更新資訊：
        - Loading：尚未取得資料
        - Success：顯示最後一次刷新時間
        - Failed：錯誤資訊
    - 內容
        - Loading：畫面空白，以 Shimmer 顯示讀取狀態
        - Success：
            - LoadingProgress 消失，顯示航班資訊列表，成功後需每十秒需刷新一次，每十秒刷新時需在畫面顯示 LoadingProgress 文字撰寫刷新中，刷新失敗時在頁面最上方，有個獨立區塊顯示錯誤訊息，但不進入錯誤畫面
            - Empty 需顯示目前無任何航班資訊
            - 橫向轉為一行兩列
            - 下拉可以打 API 即時刷新資訊
        - Failed：
            - API 錯誤時顯示錯誤訊息及重新嘗試的按鈕
